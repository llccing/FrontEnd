# HTML5版本的cnodejs

熟悉一个框架最快的方式可能就是做一个项目，所以便选了cnodejs这个社区

# cnodejs-cube (opens new window)

使用cube-ui这个库，来做个cnodejs社区的H5版本，并且结合前人做的页面效果及样式，后期功能慢慢优化。

  • https://github.com/llccing/cnodejs-cube (opens new window) 源码

# 参考

  • https://cnodejs.org/?tab=all (opens new window) -- PC版社区
  • https://github.com/lanceli/cnodejs-ionic (opens new window) -- ionic版本的cnodejs，左侧滑动效果做的很好，可以学来。
  • https://github.com/shinygang/Vue-cnodejs (opens new window) -- Vue版本的cnodejs，这个算是先驱，样式做的很好看
  • https://github.com/shuiRong/VueCnodeJS (opens new window) -- Vue PC版本重做的cnodejs社区，并且该作者分别又开发了 Nuxt、React的两个版本。赚足了star
