30.A company is using the Generative Al Security Scoping Matrix to assess security  responsibilities for its solutions. The company has identified four different solution  scopes based on the matrix. Which solution scope ST  ownership of security responsibilities? 
gives the company the MOA.Using a third-party enterprise application that has embedded generative Al  features. 
B.Building an application by using an existing third-party generative Al foundation  model （FM）。 
C.Refining an existing third-party generative Al foundation model （FM） by fine tuning the model by using data specific to the business.
D.Building and training a generative AI model from scratch by using specific data  that a customer owns. 


**D. Building and training a generative AI model from scratch by using specific data that a customer owns.**

This scope gives the company the **most ownership of security responsibilities** because it is responsible for securing the **entire model lifecycle** (data collection/handling, training pipeline, model architecture, evaluation, deployment, and ongoing monitoring), rather than relying on a third party for key parts of the stack.
