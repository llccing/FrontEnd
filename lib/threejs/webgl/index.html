# WebGL

# 简介

WebGL(Web Graphics Library)是一种JavaScript API，用于在任何兼容的Web浏览器中呈现交互式的3D和2D图形，而无需使用插件。 WebGL通过引入一个与OpenGL ES 2.0紧密相符合的API，可以在HTML5 canvas 元素中使用。

# 兼容性

https://caniuse.com/#feat=webgl
