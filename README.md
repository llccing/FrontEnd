## 🚀 快速开始

### 本地运行文档

```bash
# 安装依赖
npm install
# 或
pnpm install

# 启动开发服务器
npm run dev

# 构建静态文件
npm run build
```

### 部署

项目使用 Travis CI 自动部署，配置文件：`.travis.yml`

手动部署脚本：`deploy.sh`
